Student Attendance Risk Predictor

A machine learning project that predicts students at risk of failing based on attendance and academic performance data.

## Tech Stack
- Python
- NumPy, Pandas, Matplotlib
- Scikit-learn
- HTML, CSS
- SQL (SQLite)

Project Overview
This project builds an end-to-end ML pipeline including data preprocessing, feature engineering, model training using Logistic Regression, evaluation, and a simple dashboard to visualize predictions.
